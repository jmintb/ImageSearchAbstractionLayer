Image Search Abstractionlayer

This is an abstractionlayer for the pixabay image search api You can send a text query, paginate through results and view the latest image searchs.

Usage

Example imagesearch with offset: /imagesearch/sun?offset=2
Example get latest searches: /latest/imagesearch
